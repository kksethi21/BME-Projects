# Smart Hospital Triage System (C++)

## Overview
A C++ object-oriented system that simulates hospital triage by assigning different priority levels to patients based on
vital signs and allocating limited beds accordingly.

## Features
- Priority-based triage using vitals
- Resource (bed) management
- Inheritance

## Concepts Used
- Classes and inheritance
- Virtual functions
- Enums
- Encapsulation

## Future Improvements
- Multiple patient queue
